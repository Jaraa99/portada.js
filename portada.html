    Programacion de interfaces graficas y estructura de datos

    Profesor: Ing. Marco Guaman

    Interfaz gráfica de usuario (En inglés Graphic User Interface,
    también conocido con su acrónimo GUI) es un método
    para facilitar la interacción del usuario con el ordenador
    o la computadora a través de la utilización
    un conjunto de imágenes y objetos pictóricos (iconos, ventanas..) además de texto.
